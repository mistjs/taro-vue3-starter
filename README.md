# taro-vue3-starter
this is taro vue3 template

vue3 + unocss + auto-import + vue-components

base on [taro.3.4.x](https://taro-docs.jd.com/taro/docs/)

## Test

目前仅测试了微信小程序平台，其他平台需要自行测试

## support

* html tag
* vue3
* unocss
* auto-import
* vue-components-unplugin
* eslint-vue
* 宏函数(taro3.4.x)

## usage


```shell

# use degit

npm i -g degit

degit mistjs/taro-vue3-starter myApp

```

## install

```shell

yarn

```

### weapp
